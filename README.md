# pmm-research

A structured framework I built to run product marketing focused research across markets, competitors, and positioning.

## What this solves

Most market research is:
- descriptive
- scattered
- not directly usable for positioning

This framework turns research into:
- clear category tensions (key trade-offs in the market)
- competitor positioning breakdowns
- actionable gaps and strategic opportunities

## What it outputs

Given a company, website, and competitors, it produces:

- Market Overview  
- Challenges  
- Opportunities  
- Trends  
- Company breakdown (vision, goals, USP, positioning)  
- Competitor analysis  
- Category tension (core trade-off)  
- Positioning summary  
- Gaps  
- Non-ideal customer  
- Strategic opportunities
